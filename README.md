### üöÄ  Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/yuribmontez/ybm.news
$ cd ybm.news
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

#Set your environment variables following the .env.example structure

# Run project
$ yarn dev
```
